Detailed Description

Represents the Java class file's constant pool.

Provides the base services to manage the constant pool. The constant pool is a table which holds different kinds of items depending on their use. It can hold references for classes, fields, methods, interface methods, strings, integers, floats, longs, doubles, name and type, utf8 arrays, method handles, method types and invoke dynamic bootstrap methods.

This class works by adding these kinds of entries in an array-like structure. When an entry is removed, all entries after the one removed will be invalidated, for this reason, no removed operations are allowed.
